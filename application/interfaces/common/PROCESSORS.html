Processors

Processors are "Model" of an application, which perform the processing operation.

How do they work?

Processors are classes called inside handler methods.
They can be called once and used in all methods of a handler.
A processor will be called as a handler's property, property name is the same as with processor name.
In handler file script: 
...
public function Hello($attr){
    // Load a processor
    $this->loadProcessor("ProcessorName");
    // Use method
    $this->ProcessorName->method1($attr);
}

public function Hi(){
    // Don't need to reload processor
    // Use method
    $this->ProcessorName->method2();
}
...}
Processors must be stored in ./application/processors directory.
One PHP script file just contains only ONE processor class, the class name and the file name must be completely the same.
Their namespace must be Processor, and extend Processor class.
namespace Processor;
class ProcessorName extends Processor{
    public function method1(){
        // Do something
    }

    public function method2(){
        // Do something
    }
}
