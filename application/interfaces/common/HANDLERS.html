Handlers

Handlers are "controllers" of the application, which decide what will be executed.

How do they work?

Let's look the current URL, we're at ./Handlers.
That means we invoked the handler Handlers.
Notice: If no handler is indicated, the default one will be invoked. You can change the default in ./application/config/redirector.php.

What are they based on?

A handler is a class, stored as a PHP file script in ./application/handlers directory.
One PHP script file just contains only ONE handler class, the class name and the file name must be completely the same.
Their namespace must be Handler, and extend Handler class.
namespace Handler;
class Example extends Handler{
    // Content
}
