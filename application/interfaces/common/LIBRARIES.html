Libraries

InfinityFW also support default libraries and custom libraries.

How to use?

Libraries are classes stored in ./system/libraries. A file script can contain many classes.
They must be under namespace Library.
The following is library GetPost usage in both Handlers and Processors. 
namespace Handler;
class HandlerName extends Handler{
    public function index(){
        $this->loadLibrary("GetPost");
        $getPost = new GetPost(); // We store it in a local variable, handler property is also accepted
        echo $getPost->getPostValue("username");
    }
}
